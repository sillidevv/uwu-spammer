# uwu-spammer v1

This is a discord webhook spammer and deleter made with react, typescript and vite

## Features

- **Spamming:** Send a number of messages to a webhook

- **Webhook deletion:** Delete webhooks with a single click

- **Rate limit prevention (kinda):** Checks if the webhook has been rate limited, if so waits a little bit
